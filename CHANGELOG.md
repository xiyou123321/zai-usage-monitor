# 更新日志

## [0.0.6] - 2026-04-25

### 新增
- **模型使用占比环形图**：使用 SVG 环形图可视化各模型 Token 占比，中心显示总用量，下方图例展示各模型名称、Token 数和百分比
- **工具使用柱状图**：以彩色柱状图和数字卡片展示各 MCP 工具的调用次数
- **Token 趋势折线图**：在面板底部展示按模型分色的 Token 用量趋势图，支持 hourly（当日/7天）和 daily（30天）粒度自适应，密集标签自动旋转
- **套餐类型展示**：自动识别 API 返回的套餐类型，以标签形式显示在面板标题旁

### 改进
- **面板布局重构**：采用左右分栏设计（模型占比 + 工具统计），配额信息精简为顶部摘要条，信息密度更高
- **状态栏精简**：缩短状态栏文字格式（`T{pct}% · M{pct}%`），减少视觉干扰
- **Tooltip 精简**：优化 Tooltip 内容，减少冗余信息，保留核心指标
- **页面高度限制**：面板高度不超过可视窗口，超出部分自动滚动

### 截图
- 新增 `detail.png`：详情面板截图
- 新增 `statusbar.png`：状态栏截图
- 新增 `tooltip.png`：Tooltip 截图


## [0.0.5] - 2026-04-10

### 修复
- **修复插件重载后的重复通知问题**：使用 VS Code globalState 持久化存储通知状态，避免插件更新后重新显示已通知过的阈值
- **修复打开新窗口时的通知打扰**：启动时延迟刷新并跳过阈值检查，减少用户心智负担
- **改进用户体验**：用户主动打开使用面板时不再触发通知，仅在自动刷新或手动刷新时提醒

### 改进
- **状态栏模式配置**：新增 `statusBarMode` 配置项，支持 minimal/compact/detailed 三种显示模式
- **凭证验证**：配置凭证时自动验证有效性，提供即时反馈
- **刷新时间显示**：面板中显示上次刷新时间和下次刷新时间
- **凭证来源追踪**：显示凭证来源（Claude Code/环境变量/手动配置）
- **加载状态优化**：切换时间窗口时立即显示加载状态
- **通知持久化**：通知状态跨会话保存，避免重复提醒

### 技术改进
- 重构 `ThresholdNotifier` 类，添加状态持久化支持
- 优化插件激活流程，减少启动时的打扰
- 改进凭证配置流程，增强验证机制


## [0.0.4] - 2026-04-09

- **工具明细支持**：适配新的 `toolUsage.totalUsage` 结构，支持 `toolSummaryList` 和工具明细展示

### 改进
- **面板重构**：重新设计使用量面板，采用更紧凑的排版，优先突出 Token 配额和 MCP 配额状态
- **模块划分统一**：将页面明确拆分为配额状态、模型统计、工具统计三个区域，信息结构更清晰
- **状态栏优化**：优化状态栏文案与三种显示模式，默认信息密度更高
- **Tooltip 优化**：状态栏 tooltip 改为结构化摘要样式，展示范围、配额、重置时间和核心统计
- **交互精简**：移除导出功能与用量预测展示，减少低频功能干扰
- **文案收敛**：精简页面说明文字，保留更直接的数据标签和摘要

### 修复
- 修复模型与工具用量接口仅兼容旧字段结构的问题
- 修复面板在新旧数据结构切换时明细展示不完整的问题

## [0.0.3] - 2026-04-07

### 新增
- **智能缓存系统**：新增 CacheService，LRU 缓存策略，可配置 TTL，减少重复 API 调用
- **阈值提醒功能**：新增 ThresholdNotifier，支持多级阈值（50%/80%/95%）提醒，月度自动重置
- **增强状态栏**：全新 StatusBarManager，支持三种显示模式（详细/紧凑/精简），动态颜色变化
- **重新设计面板**：全新 UsagePanel，圆形仪表盘可视化，进度条，工具调用网格展示
- **新增配置项**：
  - `glmUsage.cacheEnabled`：启用/禁用数据缓存
  - `glmUsage.cacheTTL`：缓存有效期（秒）
  - `glmUsage.notificationThresholds`：用量阈值提醒百分比
  - `glmUsage.notificationEnabled`：启用/禁用用量阈值提醒

### 改进
- **性能提升**：智能缓存机制，相同时间窗口数据复用，减少网络请求
- **用户体验**：状态栏显示更丰富的用量信息和重置时间
- **可视化增强**：面板采用圆形仪表盘和进度条展示用量百分比
- **代码质量**：重构 extension.ts，模块化架构，新增独立服务组件
- **品牌更新**：更新扩展图标为符合 VS Code 规范的 128x128 和 256x256 PNG 格式

### 架构变更
- 新增 `src/core/CacheService.ts` - LRU 缓存服务
- 新增 `src/services/StatusBarManager.ts` - 增强状态栏管理
- 新增 `src/views/UsagePanel.ts` - 使用量面板视图
- 新增 `src/notifications/ThresholdNotifier.ts` - 阈值提醒管理
- 新增 `src/types/api.ts` 图表类型定义
